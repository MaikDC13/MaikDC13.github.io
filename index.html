¿Como Funciona?

Es un sistema de riego automatizado en el cual se riega una planta. funciona de tal manera que una vez el sensor alcance cierto nivel de humedad la bomba se dentendra y el agua dejara de fluir a la maceta, esto se repetira tantas veces el senser detecte que la maceta necesita agua y dependiendo cuanta reserva de agua tenga la bomba.

Proceso

Para realizar este proyecto se debe iniciar realizando el montaje del arduino, la pantalla LSD, el potenciometro y el sensor a la protoboard tal como esta en la imagen. acoplar la manguera con la bomba y revisar que no hayan fugas, lo siguientes es conectar la bomba al circuito. una vez este el montaje este hecho se debe conectar el arduino a una computadora para instalar el codigo que hara que funcione el motor de la bomba.
